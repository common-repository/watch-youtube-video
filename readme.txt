== Description ==

Social media tabs allows you to add YouTube subscription  to any widget area with stylish sliding tabs. Option also to have the tabs slide out from the side of the browsers. 

= Configuration Options =


* YouTube -
** Username - Enter your YouTube profile username
** Video ID - Enter a YouTube video ID if you wish to include a video in the widget




== Installation ==

1. Upload the plugin through `Plugins > Add New > Upload` interface or upload `youtube` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In the widgets section, select the youtube widget and add to one of your widget areas
4. Set the required settings and save your widget
